Prologue

Hi, I'm Syed Naimul Hasan Asif (also known as SD Asif Hossein), an undergrad student passionate about Low-level Programming, Programming Language Research, Operating Systems, Deep Learning and HPC. This blog post covers my work during Google Summer of Code 2024.

Over the time period, I have contributed to Pallene, an Ahead-of-Time compiled sister Language to Lua leveraging its most performance critical code. Although Pallene is arguably more efficient than LuaJIT (mentioned in the paper attached below) and code written in Lua C API, it lacked proper debugging support. My goal in this project was to introduce proper stack-tracing support by tracking function calls in case of runtime error enhancing overall debugging experience in Pallene.

You can learn more about Pallene in this paper, written by my Mentor Hugo Musso Gualandi.
